# relationship-ai-prompt-analyzer
Python-based relationship prompt analyzer for AI advice systems.
This is a beginner-friendly Python project that analyzes relationship-related user prompts
and categorizes them into meaningful intent groups such as advice, emotions, or questions.

## Why this project
Apps like AI relationship coaches require clean understanding of user intent.
This project demonstrates how basic prompt analysis can support AI-driven advice systems.

## Features
- Classifies relationship prompts into categories
- Uses simple Python logic (no heavy libraries)
- Easy to extend into an AI or LLM-powered system

## Example Use Case
Input: "Why do I feel ignored in my relationship?"
Output: Emotional concern / Advice request

## Tech Stack
- Python

## Future Improvements
- Add NLP libraries
- Integrate with LLM APIs
- Improve intent detection accuracy
## How it works
- User inputs a relationship-related message
- The system analyzes emotional tone and intent
- Outputs a structured summary for AI advice systems
